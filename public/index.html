Dúvidas comuns

Perguntas frequentes

O que é um arquivo DAV?
DAV é um formato de vídeo utilizado por sistemas de segurança e DVRs. É similar ao MP4, mas com um container proprietário.
Vou perder qualidade na conversão?
Não! Nosso conversor mantém 100% da qualidade original, apenas altera o container do vídeo.
Preciso instalar algo?
Não! Tudo é executado via script nativo do Windows. O FFmpeg é baixado automaticamente se necessário.
